# Screenshots

Drop the listing screenshots here as PNG. The README and the JetBrains
Marketplace listing both reference these names:

| File | What to capture |
|------|-----------------|
| `live-preview.png` | Split editor — Carve source on the left, rendered HTML preview on the right |
| `highlighting.png` | A `.crv` file showing syntax highlighting (`/italic/`, `*bold*`, `_underline_`, `~strike~`, `==highlight==`, a table, a code fence) |
| `theme-dark.png` | The live preview following the IDE dark theme |
| `settings.png` | Settings → Carve panel (PHP renderer path, custom preview CSS) |

Tips:
- Use a clean editor (hide noisy tool windows), a readable font size.
- 2x / Retina capture looks crisp on the Marketplace.
- Keep them reasonably sized (the Marketplace shows them inline).
